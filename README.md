# Logos Node Setup Presentation

A reveal.js presentation that walks through running a Logos node step by step, 
platform by platform.

## View the presentation

👉 https://**your-username**.github.io/logos-node-setup-presentation/

## How to publish your own copy

1. Fork or clone this repo
2. Push to your own GitHub account
3. Go to **Settings → Pages** and enable deployment from `main` branch `/ (root)`
4. Visit `https://<your-username>.github.io/logos-node-setup-presentation/`

## What's covered

- What is Logos and why run a node
- Platform comparison (macOS / Linux / Windows)
- Method 1: Logos Basecamp (desktop GUI — easiest)
- Method 2: Nwaku via Docker Compose (messaging node)
- Method 3: Nwaku via prebuilt binary (no Docker)
- Method 4: Logos Blockchain Node (Linux only, advanced)
- What Docker is (beginner explainer)
- Troubleshooting tips

## Local development

Open `index.html` in your browser, or serve with:

```bash
python3 -m http.server 8000
# Then visit http://localhost:8000
```
